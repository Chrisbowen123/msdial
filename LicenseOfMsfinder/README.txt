# MS-FINDER - software to predict molecular formula, class, and structure of unknown mass spectrum of EI-MS and MS/MS. It can also be used as fragment ion curation by hydrogen rearrangement (HR) rules

It is licensed under the CC-BY 4.0.
This program is managed by RIKEN and UC Davis teams. Please contact Hiroshi Tsugawa (hiroshi.tsugawa@riken.jp) for feedback, bug reports, and questions.

# Usage
See the tutorial page: http://prime.psc.riken.jp/Metabolomics_Software/MS-FINDER/index.html
And there is FAQ section as well under: http://prime.psc.riken.jp/Metabolomics_Software/MS-DIAL/index3.html
All of DLL files must be in the same folder as 'MSFINDER.exe'.

# Further
This program is directly linked with MS-DIAL. And you can efficiently use MS-FINDER if you use MS-DIAL software for data processing.
http://prime.psc.riken.jp/Metabolomics_Software/MS-DIAL/index.html
